# Practice Project

**How to open**    
  1. Download this repo.  
  2. Import mydb.sql to your database.    
  3. Setting database config on .env file.    
  4. Open the project folder.  
  5. ``` npm install```  
  6. ``` npm run dev ```  

  **Using :**
  * NodeJS  
  * Express  
  * MySQL  
  * EJS
